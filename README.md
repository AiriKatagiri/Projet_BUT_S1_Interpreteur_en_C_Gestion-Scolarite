# Projet_BUT_S1_Interpreteur_en_C_Gestion-Scolarite
Interpréteur de commande (CLI) développé en C pour la gestion des jurys de BUT Informatique. Implémente l'algorithme complexe de compensation des notes (UE/RCUE) et applique les règles de validation académique. Le projet met l'accent sur la qualité logicielle (assertions, code commenté).
